## Generation Settings Recommendations

| Setting | Value | Notes |
|---------|-------|-------|
| **Model** | Stable Diffusion XL 1.0 | Best for clean vector-like results |
| **Sampler** | DPM++ 2M Karras | Smooth gradients |
| **Steps** | 30-40 | Higher for detailed scenes |
| **CFG Scale** | 7-8 | Lower for softer results |
| **Resolution** | See individual assets | Generate at listed size |

## Post-Processing Notes

1. **Background Removal** - Use transparent background for all sprites
2. **Export Format** - PNG with transparency for sprites, JPG for backgrounds
3. **Retina Versions** - Create @2x versions for high-DPI displays
4. **File Optimization** - Use TinyPNG for web optimization
5. **Consistent Style** - Ensure all assets maintain the flat, minimalist Monument Valley aesthetic
6. **Color Harmony** - Keep colors within the pastel palette for cohesive look
